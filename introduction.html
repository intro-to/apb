Basics of HTML:

  • HTML attributes
  • Syntax: <h1></h1>,
    <h2></h2>,
    <h6></h6>, <p></p>,
    <a></a>, <br>, <img>

CSS Selectors:

  • Selectors are used to target HTML elements and apply styles to them.
  • Examples of selectors include element selectors (p, h1, div), class selectors (.classname), ID selectors (#idname), and more.

Properties and Values:

  • CSS properties define how HTML elements should be styled.
  • Each property has one or more values, which specify how the property should be applied.
  • Examples of properties include color, font-size, background-color, margin, padding, etc.

Layout:

  • CSS allows you to control the layout of a webpage, including the arrangement of elements, alignment, and positioning.
  • Techniques like floats, flexbox, and grid layout are commonly used for layout purposes.

Frameworks and Libraries:

  • CSS frameworks like Bootstrap and Foundation provide pre-designed CSS styles and components to help build websites quickly.
  • CSS libraries like Normalize.css and Reset CSS provide consistent styling across different browsers by resetting or normalizing default styles.

Key Features of Bootstrap:

  • Responsive Grid System: Bootstrap provides a responsive, mobile-first grid system that allows you to create layouts that adapt to various screen sizes and devices.
  • Pre-styled Components: Bootstrap includes a wide range of pre-styled components, such as buttons, forms, navigation bars, and cards.
  • CSS Flexbox and Grid Utilities: Bootstrap provides utility classes for CSS Flexbox and CSS Grid, allowing you to quickly align and position elements within your layout.
  • Customizable Styles: Bootstrap allows you to customize its styles to match your project's branding and design requirements using SASS variables and mixins.
